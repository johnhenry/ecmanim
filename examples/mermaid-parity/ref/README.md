# Mermaid syntax-doc corpus

The 12 diagram-type syntax documents from the official Mermaid repo
(github.com/mermaid-js/mermaid, `packages/mermaid/src/docs/syntax/*.md`,
MIT license, fetched 2026-07-10 from the `develop` branch). Each port
takes that type's FIRST full example block (the doc's own canonical
sample) as its diagram source — quoted verbatim in the port header with
the doc section it came from.
